折腾轻量级 FreeBSD 虚拟机
2023-12-30 21:00  

FreeBSD 以安全、稳定、迅速而闻名，而我的需求也很简单，DNS 服务器和内网穿透。

目标是用最小的开销稳定运行 AdGuardHome 和 ZeroTier。

开启 SSH

找到#PermitRootLogin no，取消注释并把值改成yes。
service sshd restart

配置 IPv6

vi /etc/rc.conf
添加，并将默认路由改为自己的。

重启。

启用串行控制台/serial

echo -h > /boot.config

安装 AdGuardHome

步骤有点繁琐，按照这篇文章来操作。
值得注意的是，我按照步骤做完之后发现开机自启不生效，需要再手动处理一下。
vi /etc/rc.local
添加一行
service adguardhome onestart

安装 ZeroTier

zerotier-cli join (network id)
然后就报错 401 了（
查了一下，好像这个 BUG 存在于 FreeBSD 14，解决方法：
/var/db/zerotier-one/local.conf

开启内核转发

方便通过 ZeroTier 访问局域网。

以下内容参考和修改自这位大佬。

vi /etc/pf.conf

但事实上 pf 似乎还是不会开机自启，所以还是得手动处理一下（
vi /etc/rc.local
添加一行
service pf start

启用 qemu guest agent

类似于 VMware Tools 的东西，方便和宿主机进行通讯。
